# CO1 AT2

## Capstone Title
LegalAid AI – Citizen Complaint Classification and Guidance App

This capstone project is a cloud-based application that helps citizens submit legal complaints. The system uses artificial intelligence to classify complaints and provide appropriate legal guidance. Cloud services are used to securely store complaint data, process requests, and generate reports for administrators.
